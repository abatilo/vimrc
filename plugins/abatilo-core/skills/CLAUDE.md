# Skills Directory

Files here define skills that Claude can invoke (e.g., `bd-issue-tracking`, `repo-explore`).

## Before committing changes

**Bump the plugin version.** See `../CLAUDE.md` for the checklist.

Quick reminder:
- Adding a skill → bump MINOR
- Modifying a skill → bump PATCH
- Removing a skill → bump PATCH (or MAJOR if breaking)
- Update both `plugin.json` and `marketplace.json`
